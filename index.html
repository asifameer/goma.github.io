Introduction

  • Markdown is essentially a syntax language for formatting text as you write. It’s fast, and built for people who write for the web.
  • File generated by Markdown has *.md extension.
  • Official Website
  • Learn the syntax.
  • Markdown was made by John Gruber, and he says…

Markdown is a text-to-HTML conversion tool for web writers. Markdown allows you to write using an easy-to-read, easy-to-write plain text format, then convert it to structurally valid XHTML (or HTML).

<?php
require_once 'vendor\autoload.php';
$parser = new \cebe\markdown\GithubMarkdown();
 // Get contents of markdown.md file in a var $markdown
$markdown = file_get_contents('markdown.md');
// Output html to a var $my_text
$my_text = $parser->parse($markdown);
?>

Markdown Tools

  • Add-on to export Google Doc to Markdown
    • Add-ons >> Get add-ons >> Search ‘Markdown’. Link
  • PHP parser to convert md to html
  • Download via composer or otherwise in your directory
  • Example
  • Code is
<?php
require_once 'vendor\autoload.php';
$parser = new \cebe\markdown\GithubMarkdown();
 // Get contents of markdown.md file in a var $markdown
$markdown = file_get_contents('markdown.md');
// Output html to a var $my_text
$my_text = $parser->parse($markdown);
?>
